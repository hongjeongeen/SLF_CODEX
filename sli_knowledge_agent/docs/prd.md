# Agent 기획안 (PRD) - 보험길잡이 AGENT
> 삼성생명 다이렉트 보험 가입 흐름에서 고객이 어려워하는 보험 용어를 현재 화면 맥락에 맞춰 쉬운 말로 설명하고, 비교와 상담 연결까지 이어주는 가입 보조 Agent의 MVP 기획 문서.

| 항목 | 내용 |
|---|---|
| **문제 정의** | 현재 가입 화면의 툴팁은 약관 요약 중심이라 길고 어렵다. 고객은 `통합암이 뭐지`, `일반암과 뭐가 다르지`, `금액을 올리면 뭐가 달라지지`를 알고 싶은데, 설명은 상품 문구 위주라 이해가 끊기고 설계 이탈로 이어질 수 있다. |
| **대상 사용자 / JTBD** | Primary ICP는 `암/건강보험을 실제로 비교 중인 40~50대 실수요자`. JTBD는 `지금 보고 있는 특약이 왜 필요한지, 비슷한 특약과 무엇이 다른지, 보험료를 더 낼 가치가 있는지를 30초 안에 이해하고 싶다.` |
| **핵심 기능 1** | 용어 클릭 시 쉬운 설명 제공. `통합암`, `소액암`, `비급여`, `상급종합병원` 등 핵심 용어에 연한 하이라이트를 두고, 클릭 시 `한 줄 요약 + 왜 중요한가 + 이런 분께 맞음 + 주의할 점`을 bottom sheet로 제공한다. |
| **핵심 기능 2** | 옵션 ON/OFF, 가입금액 변경 시 맥락형 설명 카드 제공. 조작 직후 `sticky guide bar`로 짧은 설명을 띄우고, `자세히 보기`를 누르면 상세 비교/설명으로 이어진다. |
| **핵심 기능 3** | 비슷한 특약 간 비교 보기. 예: `통합암 vs 일반암`, `암 치료 vs 암 통합 치료`, `비급여 포함 vs 비급여 제외`를 차이 중심으로 보여준다. |
| **핵심 기능 4** | 부담 없는 상담 연결. 설명만으로 판단이 어렵다면 `권유 없이 필요한 내용만 상담해드려요 (1588-0000)` 문구와 함께 현재 보고 있는 항목 맥락을 유지한 상담 연결을 제공한다. |
| **하지 않을 것** | 개인 건강정보 기반 추천, 상품 추천, 최적 설계 자동화, 가입 가능 여부 판단, 약관을 대체하는 법적 해석 |
| **입력 데이터** | 상품/특약 메타데이터, 용어 사전, 이벤트 카드 데이터, 비교 매핑 데이터, 화면 내 현재 선택 상태 |
| **출력 형식** | 모바일 UI 기준 `sticky guide bar`, `bottom sheet`, `비교 drawer`, `상담 연결 카드` |
| **핵심 UX 패턴** | 플로팅 아이콘 없이 `연한 하이라이트 + sticky guide bar + bottom sheet` 조합을 사용한다. 기본은 조용하게, 막히는 순간에만 개입한다. |
| **성공 지표** | 용어 설명 열람 후 다음 단계 진행률, 설명 후 옵션 선택 지속률, 해당 단계 이탈률 감소, 비교 보기 클릭률, 상담 연결 품질 |
| **HITL / 실패 처리** | 약관 기준이 중요한 질문, 모호한 보장 범위, 추천처럼 들릴 수 있는 민감한 판단은 상담으로 연결한다. 데이터가 없거나 불확실하면 단정하지 않는다. |
| **완료 정의** | 주요 암 보장 용어 20~30개에 대해 쉬운 설명이 준비되어 있고, 핵심 조작 이벤트에 대한 안내 카드가 연결되며, 최소 3쌍 이상의 비교 보기와 상담 연결 흐름이 동작한다. |

## MVP 사용자 시나리오
1. 사용자가 `통합암`을 보고 의미가 헷갈린다.
2. 하이라이트된 용어를 클릭한다.
3. 쉬운 설명과 `일반암과 비교` 액션을 본다.
4. 토글을 켜거나 가입금액을 조절한다.
5. 조작 결과에 맞는 짧은 설명 카드가 하단에 노출된다.
6. 이해가 충분하면 가입을 계속하고, 부족하면 해당 항목만 상담으로 연결한다.

## 초기 대상 용어
- 통합암
- 일반암
- 소액암
- 비급여
- 상급종합병원
- 직접치료
- 항암약물치료
- 항암방사선치료

## 우선 비교 묶음
- 통합암 vs 일반암
- 암 통합 치료 vs 암 치료
- 암 치료 vs 암 복합 치료
- 암 통합 치료 vs 비급여
- 암 고액 치료 vs 암 치료
- 주요 순환계 질환 통합 치료 vs 주요 순환계 질환 치료
- 주요 순환계 질환 치료 vs 주요 순환계 질환 복합 치료
- 주요 순환계 질환 통합 치료 vs 주요 순환계 질환 특정 치료

## 콘텐츠 작성 원칙
- `비급여`는 `국민건강보험이 적용되지 않는 치료`로 설명한다.
- `상급종합병원`은 단순히 큰 병원이라고 끝내지 않고, 자세히 보기에서 상급종합병원으로 분류된 병원 예시를 보여준다.
- 모든 상세 설명과 비교 카드에는 `자세한 기준은 약관을 꼭 확인해 주세요` 문구를 넣는다.
- 설명은 쉬운 말 중심으로 쓰되, 지급 기준과 예외는 흐리지 않는다.

## 검증 질문
1. 고객이 현재 보는 항목과 설명 카드의 맥락이 정확히 연결되는가
2. 설명이 쉬워졌지만 약관 의미 왜곡은 없는가
3. 자동 노출 카드가 방해가 아니라 도움으로 느껴지는가
4. 비교 보기와 상담 연결이 실제 선택 지속에 기여하는가
